fh_context_read ( fh_context_t * ctx,
fh_frame_readout_t * ro,

It returns true if a new fh_frame_readout_t has been successfully received and parsed. In the special case of a non blocking netwrok context, the err field should be used to distinguish case where no data is currently available (return value is false and err is FH_NO_ERROR) and end of stream cse (return value is false and err is FH_END_OF_STREAM).
